🧠 How It Works
Select a platform
Choose generation or custom wordlist
Pick username length
Choose character set
Select speed mode (Normal / Fast / Turbo)
Start scanning

Available usernames are automatically saved into text files.

🚀 Installation

Make sure you have Python installed.

Install required dependencies:

pip install requests colorama

Run the tool:

python username_checker.py
⛔ Notes
Some platforms may be slower due to rate limits
Speed depends on your internet and thread count
Press Ctrl + C to stop scanning safely
